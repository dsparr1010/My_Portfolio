Trip Buddy - A Travel Planning Resource

The application helps those planning a road trip by providing options for activities to do across multiple cities of interest along the way to the final destination. Powered by Triposo API, Leaflet Maps, Sequelize, Express and Express Session. Cravings

Think Tinder meets Yelp. Scroll through images of dishes from restaurants based on what you are craving that night. Filter for preferences and location, and you're ready to find your perfect match! The project uses the M(ySQL) E(xpress) R(eact) N(ode) Stack. The application features a user authentication system built with Passport and Bcrypt, data populated through use of Yelp Fusion API, and user-friendly React Spinners and notifications (or toasts).

A fitness tracker using Mongo database with Mongoose schema, handle routes with Express, and a responsive design using Semantic UI. The user is able to create a workout plan, add exercises to that plan, and update those exercises (to completed). The application features a stats page with readable user-friendly populated charts.
